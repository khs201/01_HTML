* 집에서 학원에서 수업한 내용(github에 push된 내용) 얻어오기 *

1. 수업한 내용을 저장할 폴더 생성 (home_workspace)

2. github 로그인(https://github.com/)

3. 원격 저장소(remote repository) 페이지로 이동

4. Code 버튼 클릭 -> clone(복제) -> 주소 복사 (https://github.com/khs201/01_HTML.git)

5. 소스트리 -> 새탭(+) -> Clone 버튼 클릭

6. 소스 경로 입력 창에 복사한 주소 붙여넣기

7. 두 번째 줄 탐색을 클릭해서 복제한 저장소의 위치를 지정

8. 선택한 위치 뒤에 "\폴더명" 을 추가 작성

9. 아래 Clone 버튼을 클릭
